# 文本转换
### Text-Converter-Android
## 好用轻便的文本高级转换工具
#### 此App支持根据正则表达式替换、删除、搜索文本，并且附带一些其他实用功能。
#### 目前已经实现的功能有：

 * 字符串搜索（可选用正则表达式匹配）
 * 字符串替换/删除（可选用正则表达式匹配）
 * 文字分段、打乱、排序（如果文字使一串数字，可根据数字大小排序）
 * AES加解密
 * Java 代码格式化、JSON格式化
 * Base64编码/解码、生成MD5摘要
 * 摩斯电码编码/解码
 * 字符串反转、分段、缩进、标号、大小写转换、批量复制
 * Markdown预览
 * 生成自定义格式的随机数、文本抽签
 * 从文件读取/写入
 * 获取网页源代码
 * HTML 格式化
 * 字符串相似度
 * 自定义正则表达式
 * 可撤销的编辑
 * 附带正则表达式的参考文档（来源：OSChina）
